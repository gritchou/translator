# Translator

This repository holds the code of a small [WebExtension](https://developer.mozilla.org/en-US/Add-ons/WebExtensions).

Translator allows user to translate text selection via context menu.
It opens a new tab on Google Translate.
Origin language is auto, Target language can be selected in a popup.
